User structure [Ref]

Field Type Description
id snowflake user id
avatar ?string user avatar hash
bio ?string user description
flags integer user flags represented as bitset
?username string username (present if user not deleted)

User flags bitset [Ref]

Value Name Description
1 << 0 (0x00000001) DELETED User is deleted (present only if user not found in database)
1 << 1 (0x00000002) STAFF employee of platform
1 << 2 (0x00000004) BOT User is bot 

GET /users/@me [Ref]

Gets currently authorized user. Example response: 
{
  "id": "2421604235608064",
  "avatar": null,
  "bio": "i like Go",
  "flags": 0,
  "username": "nerdarp"
}
